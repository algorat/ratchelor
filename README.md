A basic app :)

# Running the app

Requirements: Either yarn or npm

## With npm

Install all dependencies:

```
npm i
```

Start the app:

```
npm start
```

## With yarn

Install all dependencies:

```
yarn install
```

Start the app:

```
yarn start
```

# Building the app

The app will be built to the /docs folder.

```
yarn run build
```

or

```
npm run build
```

# Prettifying the app

```
npx prettier --write ./src
```
